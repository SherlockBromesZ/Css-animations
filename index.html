CSS Transformações, Transições e Animações

Transform

Passe o mouse sobre o quadrado para ver as transformações:

Transform
transform: translate(50px, 50px) scale(1.5) rotate(45deg);
  • translate: muda a posição nos eixos x e y
  • scale: muda a escala nos eixos x e y (maior ou menor)
  • rotate: rotaciona o elemento (pode ser em x, y, ou z)

Transition

Passe o mouse sobre o quadrado para ver a transição:

Transition
transition: background-color 1s, transform 1s 1s ease;

Você pode especificar configurações para cada estilo mudado:

  1. Nome da propriedade
  2. Duração da transição
  3. Atraso antes da execução (delay)
  4. Tipo de transição (ease, linear, ease-in, ease-out, etc.)

Animation (Keyframes)

Observe a animação do quadrado:

Animation
.animation {
    animation-name: move;
    animation-duration: 3s;
    animation-fill-mode: forwards;
    animation-delay: 1s;
    animation-iteration-count: infinite;
    animation-direction: alternate;
}

@keyframes move {
    from { transform: translateX(0); }
    to { transform: translateX(200px); }
}
  • animation-name: nome da animação definida em @keyframes
  • animation-duration: duração da animação
  • animation-fill-mode:
    • backwards: começa no estado "from" instantaneamente
    • forwards: permanece no estado "to" ao final
    • both: combina backwards e forwards
    • none: começa e termina na posição original do elemento
  • animation-delay: atraso antes de iniciar a animação
  • animation-iteration-count: número de repetições ou "infinite"
  • animation-direction:
    • normal: do "from" para o "to"
    • reverse: do "to" para o "from"
    • alternate: alterna entre "from" para "to" e "to" para "from"
    • alternate-reverse: começa com reverse e alterna

Encadeamento de Animações

Para encadear animações, você pode usar múltiplas animações separadas por vírgula:

animation: move 3s forwards, fadeOut 2s 3s forwards;

Isso executará a animação "move" por 3 segundos, seguida pela animação "fadeOut" por 2 segundos após um atraso de 3 segundos.
